# CloudPBX
This solution aim to provides a feature rich hosted PBX solution with powerful add-on Unified Communications capabilities. These also includes features such as: Pilot Number (DID) Routing, Auto attendants (Time Based Routing), IVR, Audio/Video Conferencing, Break Out Meeting Room, Public Conference, Group Ring, voicemails, Find Me, Follow Me, Sip Provisioning, Extensions. Also, CRM comes as an add-on for SME businesses includes features like, Customers Management, Opportunities, Quotes &amp; Invoices Generation, Contact Management used for number lookup, Task manager, Chat, Ticket Management for Support. This also provide a dedicated video conferencing platform with special features like Break Out Meeting Room, Scheduling etc.
